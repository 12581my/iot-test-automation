# IoT Test Automation

## 项目介绍

本项目是一个物联网设备管理平台测试工程化项目，覆盖功能测试、接口测试、UI 自动化测试、性能测试和持续集成。

## 技术栈

- Python
- Pytest
- Requests
- Playwright
- JMeter
- Allure
- GitHub Actions
- MySQL
- Linux

## 测试范围

- 用户登录
- 设备管理
- 设备状态查询
- 设备数据上报
- 告警记录
- 权限管理

## 项目结构

- test_plan：测试计划、测试用例、缺陷报告
- api_tests：接口自动化测试
- ui_tests：UI 自动化测试
- performance：性能测试
- docs：测试报告和项目总结
