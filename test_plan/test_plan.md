# IoT 设备管理平台测试计划

## 1. 项目背景

本项目模拟一个物联网设备管理平台，主要用于管理设备、查看设备状态、接收设备上报数据和处理告警信息。

## 2. 测试目标

验证平台核心功能是否正常，包括用户登录、设备管理、设备状态查询、设备数据上报、告警记录和权限管理。

## 3. 测试范围

- 用户登录
- 设备新增、编辑、删除、查询
- 设备在线/离线状态查询
- 设备数据上报
- 告警记录查询
- 权限管理

## 4. 测试类型

- 功能测试
- 接口测试
- UI 自动化测试
- 性能测试
- 回归测试

## 5. 测试环境

- 操作系统：Windows / Linux
- 数据库：MySQL
- 接口测试工具：Apifox / Postman
- 自动化测试框架：Pytest + Requests
- UI 自动化工具：Playwright
- 性能测试工具：JMeter
